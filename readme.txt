AMO.CRM Leads плагин который позволяет получать Лиды испольузая API Amo.CRM

== Description ==

AMO.CRM Leads плагин получаети выводит лиды из AMO.CRM в удобной формате.

== Installation ==

Загрузите AMO.CRM Leads плагин, активируйте его в админ-панели.
